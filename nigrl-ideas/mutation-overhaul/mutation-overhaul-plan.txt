MUTATION SYSTEM OVERHAUL — DESIGN PLAN (EXPANDED TO 80 MUTATIONS)
===================================================================
NIGRL Roguelike — gritty urban setting (Crack Den / Meth Lab)
Design goal: Replace number-soup mutations with mechanically unique identities
Inspired by: Caves of Qud physical/mental mutation design philosophy

Author note: Every mutation in this document does something. No "+1 STR" entries.
The mutation you roll should change how you play, not just tune your stats.

Version 2: Expanded from 36 to 80 mutations. Upgrade system added. New tox/rad
interaction mutations. Touch spell mutations. More proc-based effects.

I. SYSTEM OVERVIEW
================================================================

TWO TIERS (removing "huge"):
  Weak  — threshold 75+ rad, costs 50 rad, grants 100 Mutation XP
  Strong — threshold 150+ rad, costs 125 rad, grants 250 Mutation XP

Rationale for removing "huge": The existing "huge" tier was just stat
extremes (+-5 all stats, lose equipment slot). With the new system, Strong
mutations already carry equipment slot consequences as part of their design
identity. A third tier added no meaningful new design space.

Thresholds adjusted from current code values:
  Old: weak 75, strong 125, huge 250
  New: weak 75, strong 150 (bump +25 to give more weak window)

Tier selection: if rad >= 150, choose randomly between weak and strong.
If rad < 150, always weak. (Same pattern as existing code — just remove huge.)

POLARITY:
  Keep 67% bad / 33% good as the base rate.
  Favorable Odds (Mutation L3) and Five Loco drink still push this toward 50%.

PERMANENCE:
  All mutations are permanent. No expiry.
  Exception: some mutations grant permanent access to abilities that cost a
  resource per use — they are permanent grants but not free every turn.

MUTATION CAP:
  Player can accumulate any number of mutations, but a NEW flag is introduced:
    engine.mutation_count (run-wide int; already planned in branching trees doc)
  This count gates escalating hazards (see Section XII — Freak Magnet perk) and
  informs flavor text ("You are barely human anymore.").

REVERSAL:
  The Shed ability (Mutation L4) removes a random good mutation and undoes it.
  Many new mutations have unique reversal logic (see Section VII).

RAD COST DEDUCTION:
  Stays in combat.py's add_radiation() pathway. No change.

FOV MUTATION NOTE:
  This document contains 4 FOV-affecting mutations (Compound Eyes, Photosensitivity,
  Myopia, Eyeless). This is already the maximum recommended. All 44 new mutations
  in this expansion touch zero FOV. A future revision should consolidate or remove
  Photosensitivity and Myopia to reduce FOV clutter — see Section XIII.

II. UPGRADE SYSTEM — RE-ROLLING A MUTATION YOU ALREADY HAVE
================================================================

CORE RULE:
  When the mutation roller would grant a mutation the player already has,
  that mutation UPGRADES to the next level instead of granting a duplicate.

  Upgrades are tracked per-mutation in:
    engine.mutation_levels: dict[str, int]
  where the key is a snake_case mutation id string and the value is the
  current level (starts at 1 when first acquired; upgrades push to 2, 3, etc.)

  When a mutation reaches its cap level, re-rolling it has no effect (no
  duplicate, no upgrade, the roll is simply wasted). This can happen rarely.

UPGRADE PRINCIPLES:
  - Good mutations get STRONGER on upgrade.
  - Bad mutations get WORSE on upgrade.
  - Upgrades should be mechanically interesting when possible, not just +1 stat.
  - Simple mutations that can't meaningfully evolve fall back to small stat bonuses.
  - Cap levels vary: typically 3 for weak, 3 for strong (can be 2 or 4 in edge cases).

UPGRADE FORMAT IN THIS DOCUMENT:
  Each mutation entry includes an "Upgrade" block showing what each level does.
  Level 1 is the base acquisition. Level 2+ are upgrades.
  Example:
    Upgrade: L1 (base) → L2 (improved) → L3 (peak, capped)

SAVE SYSTEM:
  mutation_levels dict serializes as a JSON object (string keys, int values).
  Default for any missing key is 1 (or 0 if not acquired — only acquired
  mutations appear in the dict).

INTERACTION WITH SHED:
  Shedding a mutation removes it entirely and resets its level to 0 (removes
  the key from mutation_levels). Re-acquiring it later starts at L1 again.

INTERACTION WITH TRIPLE HELIX:
  If Triple Helix grants a bonus mutation that is the same as the triggered
  mutation, BOTH roll upgrades (the original trigger + the bonus). This can
  jump a mutation from L1 to L3 in one event. Intended — it's a rare jackpot.

INTERACTION WITH SCARRED TISSUE:
  Upgrading a bad mutation still counts as a bad mutation event — Scarred
  Tissue fires and grants +1 to a random stat. The upgrade makes it worse,
  the Scarred Tissue consolation-prizes you. Intentional tension.

--- ORIGINAL 10 (CARRIED FORWARD WITH UPGRADES ADDED) ---

1. HARDENED DERMIS — "Your skin doesn't bruise like it used to."
   Effect: Gain +2 permanent DR. You take 1 less damage from the first melee
   hit you receive on each floor (a "first hit buffer" that resets on
   floor change — tracked in engine.mut_first_hit_buffer: bool per floor).
   Upgrade:
     L1: +2 DR, first-hit buffer absorbs 1 dmg.
     L2: +4 DR total (+2 more), first-hit buffer absorbs 3 dmg.
     L3: +6 DR total (+2 more), first-hit buffer absorbs 5 dmg. Cap.
   Reversal: Remove all accumulated DR from this mutation, remove mut_first_hit_buffer.
   System hooks: player_stats.permanent_dr, engine per-floor flag.

2. ADRENALINE GLANDS — "Fight-or-flight, mostly fight."
   Effect: When your HP drops below 40%, gain Adrenaline Surge: +30 speed,
   +2 melee damage for 15 turns. This can only trigger once per floor.
   (Tracked: engine.adrenaline_surge_used: bool, reset per floor.)
   Upgrade:
     L1: HP < 40%, +30 speed, +2 melee dmg, 15t, 1/floor.
     L2: HP < 50% threshold (higher trigger window), +30 speed, +4 melee dmg, 20t.
     L3: HP < 50%, +40 speed, +6 melee dmg, 25t, effect now triggers TWICE per floor. Cap.
   Reversal: Remove the trigger entirely; effect never fires again.
   System hooks: AdrenalineSurgeEffect, combat.py HP loss path.

3. TAPEWORM SYMBIOTE — "It hurts. But you're hungry less often."
   Effect: Food items heal +3 HP on top of their normal effect.
   Upgrade:
     L1: +3 food heal bonus.
     L2: +6 food heal bonus. Drugs (strains) also grant +1 HP when used.
     L3: +9 food heal bonus. Drugs grant +2 HP. Food effects that normally have
         a chance to fail (like a bad roll on strains) cannot proc their negative
         outcomes for you — the worm filters them. Cap.
   Note: L3 strain-filtering only applies to incidental negative strain rolls
   (not to intentionally bad strains like self-damaging ones). Specifically:
   if a strain effect roll would produce a "bad roll" outcome (roll < threshold),
   re-roll once. Take the second roll regardless.
   Reversal: Remove the bonus heal, strain HP bonus, and strain re-roll.
   System hooks: player_stats.tapeworm_active, item_effects.py food heal path,
   item_effects.py strain dispatch (L3 re-roll logic).

4. COMPOUND EYES — "Everything is on fire. Also you can see all of it."
   Effect: FOV radius permanently increased by +3.
   Upgrade:
     L1: +3 FOV.
     L2: +5 FOV total (+2 more). You can see through your own venom clouds
         and Chemical Warfare effects without blocking your LOS.
     L3: +7 FOV total (+2 more). In addition, once per floor, when an enemy
         moves outside your FOV, you're notified of their direction via the
         message log ("Something moves to the northeast."). Cap.
   Note: L3 directional alert fires at most once per floor to avoid log spam.
   Reversal: Subtract mut_fov_bonus back to 0, remove extras.
   System hooks: dungeon.py compute_fov(), player_stats.mut_fov_bonus.

5. ACID SPIT — "You wheeze yellow phlegm that melts faces."
   Effect: Grants the Acid Spit ability (INFINITE charges, ADJACENT target).
   Deal 8 + (mutation_count // 3) acid damage to one adjacent enemy. Applies
   the "wet" status effect. No cost. Cooldown: 5 turns.
   Upgrade:
     L1: 8 + (count//3) dmg, ADJACENT, 5t cooldown, applies wet.
     L2: 10 + (count//3) dmg, ADJACENT, cooldown drops to 3 turns.
         Also applies "corroded" debuff (new: -1 defense for 5 turns;
         tracked as CorrodedEffect in effects.py).
     L3: 12 + (count//3) dmg, ADJACENT, 2t cooldown. Range expands to
         SINGLE_ENEMY_LOS (any visible enemy, not just adjacent). Corroded still applies. Cap.
   Note: L3 changes the ability's target_type. This requires updating the
   AbilityDef in ABILITY_REGISTRY when L3 is reached — store the current
   ability level in mutation_levels and check it on ability execution.
   Reversal: Revoke "acid_spit" entirely.
   System hooks: ABILITY_REGISTRY "acid_spit", apply_effect "wet"/"corroded".

6. EXTRA JOINTS — "Three elbows, and all of them crack."
   Effect: Movement speed permanently increases by +20. When you attack a
   stunned enemy, you gain 1 free movement step immediately after the attack.
   Upgrade:
     L1: +20 speed, free step on stunned kill.
     L2: +30 speed total (+10 more), free step triggers on ANY stunned enemy attack
         (not just kills).
     L3: +40 speed total (+10 more), free step after any stunned enemy attack AND
         you can chain up to 2 free steps per turn total (the first from a stunned
         hit, the second if a second stunned enemy is adjacent and you move to them).
         Cap.
   Reversal: Remove all speed from this mutation, remove extra_joints_active.
   System hooks: player_stats.briskness, engine.py melee attack path.

7. SUBDERMAL TOOLKIT — "Your fingers know how to break things open."
   Effect: Crates and vending machines always yield their maximum loot roll.
   You can smash a crate as a free action when you walk onto its tile.
   Upgrade:
     L1: Max loot from crates, free crate smash.
     L2: Also applies to vending machines (walking into them is free action).
         Additionally, you can pick locks on any locked doors (future content
         hook) — for now, enemy corpses always drop their maximum possible gold.
     L3: Every floor, one random crate on the map is marked "jackpot" (only you
         can detect it — shown as a slightly brighter ASCII char on your screen).
         Jackpot crates always drop 2 items instead of 1. Cap.
   Note: L3 "jackpot" crate — mark one random crate entity per floor as
   subdermal_jackpot: bool on the crate entity itself. Render as slightly
   different char (use '+' instead of '#' for the crate char).
   Reversal: Remove player_stats.subdermal_toolkit, clear any jackpot flags.
   System hooks: hazards.py crate interaction, inventory_mgr.py loot roll path.

8. REDUNDANT ORGANS — "You have two of something you didn't before."
   Effect: HoT effects tick TWICE per turn. First time you die on any floor
   in the entire run, survive with 1 HP instead.
   Upgrade:
     L1: HoT double-tick; one-time death save (run-wide).
     L2: HoT triple-tick (three ticks per turn). Death save still one-time.
         Additionally, all heal amounts (food, drinks, HoT) are increased by 10%.
     L3: HoT ticks FOUR times per turn. Heal +20% instead of +10%.
         Death save now triggers TWICE across the full run (second use tracked
         as redundant_organs_death_saves_remaining: int, default 2 at L3). Cap.
   Note: L2 and L3 flat heal bonuses stack with Tapeworm Symbiote's bonus.
   Reversal: Remove HoT multi-tick (reset to 1 tick), remove death save(s),
   remove heal bonus.
   System hooks: effects.py HoT tick path, engine.py player death check.

9. NEURAL STATIC — "Your brain hums. Enemies get a headache."
   Effect: All enemies within 3 tiles of you have their crit chance reduced
   by 10 (minimum 0). This is an aura applied passively each turn.
   Upgrade:
     L1: -10 crit to enemies within 3 tiles.
     L2: -15 crit to enemies within 5 tiles.
     L3: -20 crit within 5 tiles. Additionally, enemies that crit you anyway
         (if they have naturally very high crit rates) are briefly stunned for
         1 turn after landing the crit (neural feedback). This stun is flagged
         as neural_feedback_stun: bool on the attacker entity temporarily. Cap.
   Reversal: Remove neural_static_active flag.
   System hooks: combat.py enemy attack resolution.

10. VENOM NODES — "Something's been added to your saliva."
    Effect: Melee attacks always apply 1 stack of venom on hit (no chance roll).
    Upgrade:
      L1: 1 venom stack on every melee hit.
      L2: 2 venom stacks on every melee hit. Additionally, venom from this
          mutation lasts 1 extra turn per stack (effect duration +1).
      L3: 3 venom stacks on every melee hit. Venom from this mutation now ticks
          for 2 damage per stack instead of 1 (stored in the VenomEffect as an
          upgraded_venom: bool flag). Cap.
    Note: L3 upgraded venom only applies to stacks from this mutation. Weapon
    venom stacks remain at 1 dmg/stack/turn. Track which stacks are upgraded
    via the venom application source — simplest implementation: L3 venom
    nodes just apply a separate "venom_upgraded" effect that stacks independently.
    Reversal: Remove venom_nodes_active flag and ability to apply.
    System hooks: combat.py melee hit resolution, effects.py VenomEffect.

11. TOUCH OF FROST — "Your hands are always cold. Now they share it."

13. TOUCH OF SHOCK — "Static electricity, but personal."

      L1: 3 dmg, 2 shocked stacks, 4t cooldown.
      L2: 5 dmg, 3 shocked stacks, 3t cooldown. Enemies with 4+ shocked stacks

14. IRON GUT — "You can eat anything. Anything."
    Effect: Using drugs (strains) never deals self-damage, even on bad rolls.
    Additionally, when you consume a drug, you gain 3 temporary HP that lasts
    5 turns (TempHP_IronGut, stacks up to one active instance — refreshes on
    next drug use). Does not interact with food, only strains.
    Upgrade:
      L1: No strain self-damage; +3 temp HP on drug use (lasts 5 turns).
      L2: No strain self-damage; +6 temp HP on drug use (lasts 8 turns).
          Tolerance gains from drug use are doubled (Munching XP per use × 2).
      L3: No strain self-damage; +10 temp HP on drug use (lasts 12 turns).
          Tolerance gains doubled. Additionally, strains that roll "bad" outcomes
          (below threshold) are converted to "neutral" outcomes — no beneficial
          effect but also no penalty. Cap.
    Note: L3 "neutral" conversion means the 1-39 bad range on strain rolls
    produces a "you feel nothing unusual" message instead of a penalty.
    Reversal: Remove iron_gut_active flag. Remove temp HP if currently active.
    System hooks: player_stats.iron_gut_active, item_effects.py strain dispatch
    (block self-damage, apply temp HP, L3 bad-roll conversion).

15. SCAR TISSUE BUILDUP — "You stop bleeding where you used to bleed."
    Effect: You gain 10% resistance to bleed and venom effects (they still
    apply but their damage is reduced by 10%). Additionally, your melee hits
    have a 20% chance to apply 1 stack of bleed to the target even without a
    bleed weapon. Flavor: your scarred, ragged fists tear enemy skin.
    Upgrade:
      L1: 10% bleed/venom resistance; 20% on-hit bleed chance (1 stack).
      L2: 20% resistance; 30% on-hit bleed chance (2 stacks).
      L3: 30% resistance; 40% on-hit bleed chance (2 stacks, longer duration:
          bleed from this mutation lasts 8 turns instead of 5). Cap.
    Note: This bleed is independent of weapon on_hit_bleed. They stack.
    Reversal: Remove scar_tissue_buildup_active flag.
    System hooks: player_stats.scar_tissue_buildup_active, combat.py melee hit
    (chance bleed apply), effects.py bleed damage (check resistance flag).

16. PHEROMONE GLAND — "You're more attractive than before. In a bad way."
    Effect: Dark humor. Enemies within 4 tiles are compelled to target you
    before other valid targets (if an enemy is in range of multiple entities,
    it prioritizes you). HOWEVER, enemies within 2 tiles of you suffer -2
    to their effective melee power (pheromones confuse them on impact).
    Upgrade:
      L1: Pheromone attraction (4 tiles priority), -2 enemy power at <= 2 tiles.
      L2: Attraction range stays 4 tiles. Enemy power penalty increases to -4
          within 2 tiles. Enemies that miss you due to the confusion have a
          15% chance to waste their action (they attack the air instead).
      L3: -6 enemy power within 2 tiles. Enemies within 1 tile have a 25% chance
          to skip their attack entirely (completely confused by the scent). Cap.
    Note: The targeting priority is implemented in ai.py: if pheromone_gland_active
    and player is within 4 tiles, set target to player even if another entity
    is closer. The power penalty is in combat.py enemy attack resolution.
    Reversal: Remove pheromone_gland_active flag.
    System hooks: player_stats.pheromone_gland_active, ai.py target selection,
    combat.py enemy melee power calculation.

17. RAD ABSORPTION — "The radiation. You're... processing it now."
    Effect: Each time you gain radiation, you convert 10% of the gained amount
    into bonus melee damage for your next attack (rounded down, minimum 1, capped
    at +10). This bonus resets after the next melee hit fires.
    Flavor: the rad spikes through your fists on impact.
    Upgrade:
      L1: 10% of rad gain → melee damage bonus, cap +10, resets on hit.
      L2: 20% conversion, cap +15. The bonus also applies to Touch of Frost/Fire/Shock
          abilities (if you have them).
      L3: 30% conversion, cap +20. When the bonus fires, you also irradiate the
          target for 5 rad (minor contamination on impact). Cap.
    Tracking: engine.rad_absorb_bonus: int (resets to 0 after any melee attack).
    Note: This mutation makes radiation into a melee resource instead of pure hazard.
    Reversal: Remove rad_absorb_active flag, clear rad_absorb_bonus.
    System hooks: player_stats.rad_absorb_active, combat.py add_radiation() for
    conversion calculation, combat.py player melee hit (consume bonus + L3 irradiate).

18. CHITINOUS HIDE — "Something grew between your shoulder blades."
    Effect: Gain +1 permanent DR. Whenever you receive 3 or more consecutive melee
    hits in a single floor combat (tracked as chitin_hit_streak: int per floor),
    you gain 1 stack of Hardening (new: HardeningEffect, +1 DR per stack, max 5
    stacks, expires when combat ends — tracked as no enemies in LOS for 5 turns).
    Upgrade:
      L1: +1 DR base; Hardening stacks start at 3-hit threshold; max 5 stacks.
      L2: +2 DR base. Hardening starts at 2-hit threshold. Max 7 stacks.
      L3: +3 DR base. Hardening starts on 1st consecutive hit (every hit stacks
          it). Max 10 stacks. Hardening stacks take 1 extra turn to fade (10t
          no-combat timer instead of 5t). Cap.
    Tracking: engine.chitin_hit_streak: int (increments on melee hit received,
    resets to 0 if player makes a melee attack that lands without taking a hit
    that turn — i.e., player hits first in that turn).
    Reversal: Remove chitinous_hide_active, remove DR, clear any Hardening stacks.
    System hooks: player_stats.permanent_dr, effects.py HardeningEffect,
    combat.py incoming melee (streak tracking, apply effect).

19. LEECHING STRIKE — "You pull something out of them when you hit."
    Effect: Melee hits on enemies with 3 or more active negative status effects
    (venom, bleed, chill, shocked, ignite, corroded, wet — any combination)
    grant you 2 HP per hit (capped at your max HP). This is a small leech on
    debuffed targets. Not regen — only fires when conditions are met and you
    actively hit.
    Upgrade:
      L1: 2 HP leech on hitting enemies with 3+ debuffs.
      L2: 3 HP leech; threshold drops to 2+ debuffs.
      L3: 4 HP leech; threshold drops to 1+ debuff (any debuffed target triggers it).
          Additionally, if the target dies from a hit that triggers leech, the
          leech amount doubles (8 HP on kill while target is debuffed). Cap.
    Reversal: Remove leeching_strike_active flag.
    System hooks: player_stats.leeching_strike_active, combat.py melee hit
    resolution (count negative effects on target, apply heal if threshold met).

20. BURST CAPILLARIES — "When you bleed, things get messy for everyone."
    Effect: When YOU take bleeding damage (from any bleed effect), blood sprays
    outward — each bleeding tick has a 40% chance to apply 1 bleed stack to any
    enemy adjacent to you. Flavor: your blood, under pressure from the mutation,
    sprays outward on every bleed tick.
    Upgrade:
      L1: 40% per bleed tick → spray 1 bleed stack to adjacent enemies.
      L2: 60% per tick, spray 2 stacks to adjacent enemies. Range extends to 2 tiles.
      L3: 80% per tick, spray 3 stacks. Range 2 tiles. Enemies hit by blood spray
          take 1 direct damage in addition to the bleed stack. Cap.
    Note: This turns a bad status (being bled) into a weapon. Synergizes with
    Second Skin of Thorns (which makes you take self-damage on attack, potentially
    combined with a bleed weapon to self-apply bleed). Classic NIGRL risk/reward.
    Reversal: Remove burst_capillaries_active flag.
    System hooks: player_stats.burst_capillaries_active, effects.py BleedEffect.tick()
    on player (spray check), combat.py entity adjacency helper.

IV. WEAK BAD MUTATIONS (20)
================================================================

Design principle: Bad mutations should change how you play, not just hurt you.
Even purely negative mutations should create interesting decisions.

--- ORIGINAL 10 (CARRIED FORWARD WITH UPGRADES ADDED) ---

1. ROTTEN TEETH — "They just come out now."
   Effect: Food items take 1 additional turn to eat (2 turns of energy instead
   of 1). Food heals 3 less HP (minimum 1).
   Upgrade (gets worse):
     L1: +1 turn eating; -3 food heal.
     L2: +2 turns eating; -5 food heal (minimum 1). Eating any food now makes
         a loud noise (enemies within 5 tiles are alerted by the crunching).
     L3: +3 turns eating; -7 food heal (minimum 1). The noise alert radius
         extends to 8 tiles. Food items also have a 15% chance to deal 1 dmg
         to you on consumption (biting yourself). Cap.
   Reversal: Remove all penalties, remove noise-alert behavior.
   System hooks: rotten_teeth_level: int, foods.py eating duration, item_effects.py
   food heal, engine.py eat-action enemy alert trigger.

2. PHOTOSENSITIVITY — "Direct light burns. Not metaphorically."
   Effect: FOV radius reduced by -2. Edge-of-FOV tiles flicker (every other turn
   they don't update).
   Upgrade (gets worse):
     L1: -2 FOV, edge flicker.
     L2: -4 FOV total. Flicker extends to the outer 2 tiles of FOV. Additionally,
         being within 3 tiles of a fire hazard causes 1 damage per turn (light pain).
     L3: -6 FOV total. Flicker on outer 3 tiles. Fire proximity damage increases
         to 2/turn at 2 tiles, 3/turn at 1 tile. Cap.
   Note: This is the weakest FOV mutation at L1, but L3 becomes genuinely harsh.
   Reversal: Remove all fov_penalty, remove fire proximity behavior, remove flicker.
   System hooks: player_stats.photosensitivity_level: int, dungeon.py compute_fov,
   render.py FOV update, engine.py per-turn fire proximity check.

3. REACTIVE SKIN — "Touch anything and it's gonna be a whole thing."
   Effect: Grease tiles deal 3 damage per step. Fire tiles deal +50% more damage.
   Upgrade (gets worse):
     L1: Grease 3 dmg/step; fire +50%.
     L2: Grease 5 dmg/step; fire +100%. Also: venom from enemies applies 1 extra
         stack beyond their normal amount (skin absorbs more).
     L3: Grease 7 dmg/step; fire +150%. Venom +2 extra stacks. Additionally,
         the wet status effect (normally neutral) causes 1 damage per tick on you
         specifically (wet + reactive skin = chemical irritation). Cap.
   Reversal: Remove reactive_skin flag and all its conditional checks.
   System hooks: reactive_skin_level: int, engine.py tile-damage resolution.

4. VESTIGIAL LIMB — "Your left arm is just vibes now."
   Effect: -1 melee power permanently.
   Upgrade (gets worse):
     L1: -1 melee power.

    Upgrade (gets worse):
      L1: Movement costs 125 energy; item pickup costs 1 energy.
      L2: Movement costs 150 energy; pickup costs 5 energy. Additionally,
          knockback effects push you 2 extra tiles (more susceptible to force).
      L3: Movement costs 175 energy; pickup costs 10 energy; knockback +3 tiles.
          Additionally, abilities that target you for teleport/pull effects (like
          Telekinetic Lash if the player ever faces such things from enemies) are
          1.5× as effective on you. Cap.
    Note: Gravity Sensitivity + Extra Joints (weak good +40 speed at L3) is
    an interesting balance — the extra joints might offset some of the movement
    cost but not all of it.
    Reversal: Remove gravity_sensitivity_active flag and movement cost modifier.
    System hooks: player_stats.gravity_sensitivity_active, engine.py move action
    (energy cost modifier), inventory_mgr.py item pickup (energy cost).

19. PERMEABLE SKULL — "Things get in. They don't get organized correctly."
    Effect: Enemy abilities that would normally be blocked by Redundant Brain
    immunity or any future crowd-control immunity items penetrate through anyway.
    Additionally, all crowd-control effects (confuse, fear, stun, slow) applied
    to you last 2 extra turns. Your skull doesn't protect the brain from impact.
    If you don't have Redundant Brain, the +2 turn duration penalty still applies.
    Upgrade (gets worse):
      L1: Crowd-control immunity bypass; CC effects last +2 turns.
      L2: CC effects last +4 turns. Additionally, when you are stunned, you
          also take 1 damage per turn while stunned (panic damage).
      L3: CC effects last +6 turns. Stun panic damage increases to 3/turn.
          Additionally, the fear effect, when applied, causes you to run for
          an extra direction even after it expires — 1 additional turn of
          fear-movement happens post-expiry (phantom fear). Cap.
    Reversal: Remove permeable_skull_active flag.
    System hooks: player_stats.permeable_skull_active, effects.py apply() for
    all CC effects (bypass immunity check, extend duration).

20. FISSION HEART — "It beats once. Then it irradiates everything within five feet."
    Effect: Your heart has partially fissioned. On every combat turn (turn where
    you deal melee damage or take melee damage), your heart pulses and emits
    20 rad to YOU and 5 rad to all enemies within 3 tiles. This happens after
    the combat action resolves. The self-rad is real and can trigger mutations.
    Flavor: the cost of having a nuclear heart.
    Upgrade (gets worse):
      L1: Combat turn → you gain 20 rad; enemies within 3 tiles gain 5 rad.
      L2: You gain 30 rad; enemies gain 10 rad; radius expands to 4 tiles.
      L3: You gain 40 rad; enemies gain 15 rad; radius 5 tiles. Additionally,
          the pulse fires every turn regardless of combat (not just on attack/take-damage
          turns) — passive continuous irradiation. Cap.
    Note: At L3 this is the strongest self-rad mutation in the game. Combined
    with Radiation Sponge (strong bad), a player would self-irradiate at an
    extreme rate. This creates volatile mutation cascades. Intentional risk.
    Reversal: Remove fission_heart_active flag.
    System hooks: player_stats.fission_heart_active, engine.py per-turn (L3)
    or combat.py (L1/L2) post-combat hook (pulse logic).

VII. REVERSAL (SHED ABILITY) — UPDATED LOGIC
================================================================

The Shed ability (Mutation L4) remains unchanged in concept:
  - Pick a random GOOD mutation from mutation_log
  - Undo it (call reversal logic)
  - Remove a random debuff
  - Refund half the tier's rad cost

Also: Shedding a mutation resets engine.mutation_levels[mutation_id] to 0
(removes the key from the dict).

New reversal types needed (beyond existing all_stats/single_stat/resistance/
skill_points/skill_level/briskness/dr/lose_slot/lose_5_skills):

  "grant_ability": revoke the named ability via engine.revoke_ability(ability_id)
  "flag": set player_stats.{flag_name} = False
  "speed": reverse a permanent speed delta on engine.player.speed
  "fov_bonus": subtract from player_stats.mut_fov_bonus
  "power_penalty": add back the lost power to engine.player.power baseline
  "slot_block": remove the named slot from blocked_equip_slots
    (new: player_stats.blocked_equip_slots: set[str])
  "compound": list of sub-reversals, each processed in order
  "ability_and_flag": shorthand for revoke ability + clear flag in one step
  "stat_and_flag": shorthand for stat reversal + clear flag

These reversal types extend the undo_mutation() function in mutations.py.
The reversal dict for compound mutations looks like:
  {"type": "compound", "steps": [{"type": "flag", "flag": "carapace_active"},
                                   {"type": "dr", "amount": 5},
                                   {"type": "slot_block", "slot": "hat"}]}

Save system note: All new player_stats flags (booleans) and
player_stats.blocked_equip_slots (set) must be added to save_system.py
serialization. Use JSON-serializable types (list for set).

UPGRADE REVERSAL RULE:
  When Shed removes a mutation, the full effect of ALL its upgrade levels is
  reversed in one step. The reversal functions must check mutation_levels[id]
  to know how much to undo.
  Example: Hardened Dermis at L3 has granted +6 DR. Shed must remove 6 DR,
  not 2 DR. The reversal dict for leveled mutations should use
  {"type": "compound", "steps": [dynamically computed from mutation_levels[id]]}.
  In practice: mutations.py undo_mutation() fetches the mutation's current level
  from mutation_levels, then calls the reversal logic scaled to that level.

VIII. SCARRED TISSUE INTERACTION
================================================================

Scarred Tissue (Mutation L1): "Bad mutations also grant +1 to a random stat."
This perk fires apply_scarred_tissue() in mutations.py after any bad mutation.

With the new system, all 20 bad weak and 20 bad strong mutations are still
"bad" polarity. Scarred Tissue fires on all of them, including upgrades.
Upgrading a bad mutation counts as a bad-mutation event — you get your +1 stat
consolation even as the mutation worsens.

Special interaction — Degenerating Nervous System (Weak Bad #20):
  Acquiring any mutation (good or bad) increments the degen counter. Scarred
  Tissue fires and grants +1 stat on bad mutations, but the nervous system
  immediately adds 1 more turn to all ability cooldowns. Every bad mutation
  is now a triple-effect: bad mutation, +1 stat from Scarred Tissue, +1 ability
  cooldown from Degen Nervous System. This is hell in the best possible way.

Special interaction — Accelerated Aging (Strong Bad #14):
  Scarred Tissue fires on Accelerated Aging acquisition (+1 stat). But every
  subsequent mutation now also removes 1 max HP. Scarred Tissue still fires on
  each bad mutation — you get +1 stat and lose 1 max HP simultaneously. The
  tradeoff is sharp.

Special interaction — Fission Heart (Strong Bad #20):
  Scarred Tissue fires. Player gets +1 stat. But now every combat turn irradiates
  you, which can trigger more mutations, which trigger more Scarred Tissue procs,
  which is its own chaotic cascade. High mutation builds become radiation-fueled
  stat engines with increasingly dangerous side effects.

Existing Mutation skill perks (L1-L5) continue to work as designed:
  L1 Scarred Tissue — bad mutation → +1 random stat (fires on upgrades too)
  L2 Unstable — 20% on melee hit: Unstable buff (unchanged)
  L3 Favorable Odds — +50% good mutation multiplier (unchanged)
  L4 Shed — reversal logic extended (see Section VII)
  L5 Triple Helix — 30% bonus mutation on any mutation (can trigger upgrade +2 levels)

Reference values from MEMORY and current code:
  Player HP: 30 + CON*10. At CON 10: 130 HP.
  Typical floor 1-2 enemy damage: 4-8 per hit.
  Swagger defense: int((swagger-8)/2). Default 0.
  Strong enemy (floor 3-4): ~60-80 HP, 8-12 damage.
  Rad thresholds for mutation: 75 (weak) / 150 (strong).

Weak Good power targets (L1 = "slightly better than +1 stat"):
  - Touch of Frost/Fire/Shock: 3-8 damage + elemental status, ADJACENT, 1-4t cooldown.
    At L1 these add roughly 1-3 DPS over time. Similar to +1 STR. Fine.
  - Leeching Strike (2 HP leech on 3+ debuff hits): good synergy with debuff builds,
    minimal impact in isolation. Won't be active early game. Weak-tier correct.
  - Burst Capillaries (40% bleed spray on bleed ticks): dependent on already having
    bleed applied to yourself. Conditional — weak-tier appropriate.
  - Iron Gut (no strain self-damage, +3 temp HP): very build-specific. Drug-heavy
    players love it, melee-only players don't care. Weak-tier correct.

Weak Bad pain targets (L1 = "annoying but survivable"):
  - Porous Membrane (+5 tox per corpse step): In Crack Den with moderate kill
    counts, player might step on 5-10 corpse tiles per floor. That's 25-50 tox.
    Annoying but manageable. Meth Lab: much worse (more corpses). Monitor.
  - Phantom Hunger (-4 HP every 30 turns): roughly 0.13 DPS. On a 100-turn floor,
    that's 13 HP. At default 130 HP, that's 10%. Fine for weak tier.
  - Slow Blink (no items/abilities/guns for 1 turn on combat start): pure combat
    nuisance. 1-turn lockout is forgiving. Weak-tier appropriate.
  - Involuntary Bioluminescence (all enemies always know position): similar to
    territorial marking but through walls. This is strong for weak tier — reconsider
    if playtesting shows it's effectively run-ending on Meth Lab stealth floors.

Strong Good power targets (build-defining, worth 125 rad and 250 mutation XP):
  - Irradiated Blood (rad splash when hit): turns "taking hits" into "irradiating
    nearby enemies." Good survivability tool in tight corridors. Strong-tier correct.
  - Feral Instincts (melee kill → Feral Rush, +45 speed/+6 power at L3): very
    powerful in clear-room scenarios. Limited in 1v1. Strong-tier correct.
  - Scavenger Metabolism (on kill: absorb tox + heal): heals ~4 HP per kill at L2.
    In a 20-enemy floor this is 80 HP of healing over the floor. Significant.
    Balanced by being all-tox-source: you gain real tox from each kill.
  - Second Stomach (×3 intensity, ÷2 duration at L3): this is potentially very
    powerful for speed strains (triple speed, half duration). The neck slot cost
    is meaningful. May need monitoring for drug-rush builds.

Strong Bad pain targets (run-altering but not run-ending):
  - Fission Heart (L3: 40 self-rad + 15 enemy rad per turn): at L3, this is
    ~2.4 self-rad per average turn (if player acts every ~17 ticks). In a 100-turn
    floor, approximately 240 rad accumulated. This alone would trigger 2-3 mutations
    per floor. Combined with Radiation Sponge: devastating. Run-altering at L3. Good.
  - Degenerating Marrow (-2 power total + -10 speed + marrow flares at L3): the
    power penalty is moderate (-2 on floor 4 enemies is -6.6% damage reduction).
    The speed penalty matters more. Strong-tier appropriate.
  - Accelerated Aging (-1 max HP per mutation, cap 25% of base): at max mutations
    (say 10 mutations on a CON 8 build: base 110 HP), cap = 27 HP (25% of 110).
    Player could lose up to 83 HP of max HP from mutations alone. This is severe
    late-game. Strong-tier correct. Intended to discourage spam mutation builds.

  - Leeching Strike + Bone Spurs: hitting an enemy applies bleed (30%+), creating

  - Muscle Memory Override + Twitchy Nerves: counterattacks from MMO cannot miss

L2 — Unstable: 20% on melee hit to gain Unstable buff (+5 rad, +2 dmg, irradiate).
  With venom_nodes_active (all levels) every melee hit also applies venom.
  With Feral Instincts active (FeralRush), melee hits are more frequent →
  more Unstable procs. No conflicts.

L3 — Favorable Odds: raises good% from 33% to ~50%. With 20 good/20 bad in each
  tier (40 good, 40 bad total), Favorable Odds shifts the coin. Still meaningful.

L4 — Shed: reversal logic extended (see Section VII). ALL new mutations must
  have valid reversal dicts. Upgrade scaling must be reversed at the correct level
  (e.g., Hardened Dermis at L3 reverses 6 DR, not 2). The undo_mutation()
  function fetches mutation_levels[id] before reversing.

L5 — Triple Helix: 30% chance for bonus mutation after any mutation. With level
  system, this can jump a mutation from L1 to L3 if the same mutation is rolled
  twice consecutively (original trigger + bonus both upgrade the same mutation).
  In edge case: player has Carapace L1 and rolls Carapace → L2. Triple Helix
  bonus rolls Carapace again → L3. Player has full Carapace in one event.
  This is a rare jackpot. Intended. Hat slot remains blocked throughout.

Future L6+ Mutation perk ideas (not in scope, noted for later):
  - "Directed Mutation": Once per floor, choose polarity (good/bad) for next roll.
  - "Splice": Merge two weak mutations into one randomly selected strong mutation.
  - "Stable Genome": All future mutation upgrades apply at L+2 instead of L+1
    (skip levels; reach cap faster but fewer total mutation events).

XIII. DESIGN SCRATCHPAD — CUT IDEAS AND FUTURE DIRECTIONS
================================================================

FOV MUTATION AUDIT:
  Current FOV mutations: Compound Eyes (+3/5/7 FOV), Photosensitivity (-2/4/6 FOV),
  Myopia (-3/5/7 FOV), Eyeless (raw FOV 4/5/6 + sonar). That is FOUR FOV mutations.
  Recommended future action: Remove or redesign Photosensitivity and Myopia in a
  future revision to reduce FOV-touching mutations from 4 to 2 (Compound Eyes + Eyeless).
  The mechanical space of "+/- FOV" is thin and these mutations are less interesting
  than their effect description suggests. Consider replacing:
    Photosensitivity → something that modifies what you can DO at distance, not just see.
    Myopia → something that creates a movement or targeting constraint, not just range.
  For this revision, they are kept as-is per the "keep existing 10" rule.

CUT: "Regrowth" — heal 1 HP per turn passively. Cut per project rules: no passive
  per-turn regen. Players exploit by waiting.

CUT: "Telepathy" — gave perfect knowledge of enemy intentions (next action).
  Cut because NIGRL's AI is simple enough that this is not interesting —
  most enemies just chase and attack. Revisit if AI gets action queues.

CUT: Third eye / psychic variants — too sci-fi. Neural Static kept because
  it's passive and invisible. Psionic Echo kept because it's tied to ability use.

CUT (from new strong good ideas): "Bone Wings" — flight over grease/fire.
  Replaced by Vestigial Wings (partial flight only) which is more thematically
  appropriate and doesn't negate entire hazard systems.

CUT (from new strong bad ideas): "Permanent Blindness" — -10 FOV, sonar only.
  Too similar to Eyeless. Cut.

CUT (from new weak bad ideas): "Static Electricity" — 20% chance to drop
  metal weapons when hit. Too punishing + item-interaction complexity is high
  for weak tier. Replaced by Lopsided Growth (stumble mechanic, cleaner).

FUTURE: "Mutation synergy" system — if a player has 3+ mutations from the
  same "theme" (body mutations: Carapace + Bone Spurs + Hardened Dermis),
  they unlock a special bonus passive. This would be a Mutation L6+ perk.

FUTURE: Monster-specific mutations. The monster mutation system (already in
  mutations.py) could be overhauled similarly — replace stat-only mutations
  with behavioral ones (Sticky Tendons = monster grapples player on hit,
  Explosive Death = monster detonates when killed).

FUTURE: Mutation cooldown display — when acid_spit is on cooldown, show
  remaining turns in the ability list. Current ability system doesn't show
  cooldowns. This is a render.py / menu_state.py concern.

FUTURE: Named mutation combinations. If a player ends up with Fission Heart
  + Irradiated Blood + Rad Absorption simultaneously, the mutation log could
  display "Nuclear Physiology (3-part)" as a special achievement flavor tag.
  No mechanical difference — purely cosmetic/flavor.

BALANCE FLAG: Bioluminescence (Weak Bad) may be effectively stronger than
  some Strong Bad mutations. If playtesting shows it's consistently more
  run-ending than Freak Magnet (which has hard numbers), consider bumping it
  to Strong Bad and replacing it with a genuinely weak detection-range tweak.

BALANCE FLAG: Second Stomach (Strong Good, neck blocked) is potentially
  very powerful for movement-speed drug builds. If speed strains become
  meta-defining in combination with this, consider capping the intensity
  multiplier at ×2 for speed values specifically, keeping ×3 for other stats.
