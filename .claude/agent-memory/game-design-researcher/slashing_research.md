Slashing niche defined as SWEEP (multi-target melee AOE) + LACERATION (movement-punishing bleed DoT).

**Why:** Beating owns stun/knockback, Stabbing owns single-target burst/Windfury, Smacking owns unarmed stun chains. Multi-target melee damage is unowned. Movement-punishing bleed creates unique synergies with knockback (Beating) and counter-synergy with stun (Stabbing Gouge).

**How to apply:** Shared trunk is Cleave (L1) -> +3 STR (L2) -> Laceration passive (L3). LacerationEffect uses independent timers like GlassShardsEffect. 10 branching tree designs at L4+ provide different specializations. Top 3 recommended: Design 1 (Butcher: AOE vs DOT), Design 10 (Warden: 3-way hunt/defend/spread), Design 3 (Surgeon: debuff vs alpha-strike).

Key design patterns discovered across 14 games:
- PoE/CDDA: movement-punishing bleed = unique in turn-based roguelikes
- Diablo: sword = crit amplification; Rend = AOE applied bleed; D4 bleed = vulnerability state
- Dark Souls/Elden Ring: threshold bleed (Hemorrhage) = build stacks to burst; Iaido = delayed high-damage counter
- Monster Hunter: Spirit Gauge = momentum tier system (consecutive hits = persistent damage buff that decays)
- Slay the Spire: alternative stat scaling (3x multiplier for one attack); "spend all for proportional power"
- TOME4: 4-tier perk progression pattern (notice -> button -> shift -> capstone)

10 design patterns identified: Cleave/AOE, Bleed/DOT, Movement-Punishing Bleed, Momentum/Frenzy, Riposte/Counter, Threshold Burst, Speed/Energy, Execute, Debuff/Cripple, Iaido/First Strike.
